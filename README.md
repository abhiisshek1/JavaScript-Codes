# JavaScript - Getting Elements by ID

This repository contains two JavaScript examples that demonstrate how to access and manipulate HTML elements using the `getElementById()` method.

## 📌 Description

The purpose of this repository is to help beginners understand how JavaScript interacts with HTML by selecting elements using their unique IDs. The examples show how to retrieve an element, modify its content, and respond to user actions such as button clicks.

## 📂 Files

- **index.html** – Contains the HTML structure with elements and buttons.
- **script1.js** – Demonstrates selecting an element using `document.getElementById()` and changing its content.
- **script2.js** – Shows another example of accessing elements by ID and performing different DOM manipulations.

## 🚀 How to Run

   ```bash
   git clone https://github.com/your-username/your-repository-name.git
   ```

## 💡 Example

```javascript
function changeText() {
    document.getElementById("msg").innerHTML = "Hello, JavaScript!";
}
```
